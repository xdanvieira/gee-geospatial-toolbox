# GEE Geospatial Toolkit

Reusable Google Earth Engine scripts for hydrology, remote sensing, vegetation monitoring, climate analysis and geospatial applications.

## Modules

| Folder | Description |
|---|---|
| `hydrology/` | Hydrological workflows: precipitation, runoff, evapotranspiration, water deficit, CWSI and SEBAL. |
| `satellite/` | Satellite preprocessing and visualization workflows. |
| `vegetation/` | Vegetation indices and biomass-related workflows. |
| `climate/` | Climate datasets and drought indicators. |
| `utilities/` | Export, visualization and helper functions. |
| `examples/` | Applied examples and case studies. |
